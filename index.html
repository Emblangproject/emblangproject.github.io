Emblang - Simple Educational Programming Language

Project Introduction

Emblang is a very simple programming language designed to teach basic compiler and programming language concepts. It allows writing arithmetic expressions and print statements with minimal syntax.

Features

  • Supports arithmetic operators: addition (+), subtraction (-), multiplication (*), and integer division (/)
  • Supports parentheses for expression precedence
  • Print statement for outputting values
  • Detects errors such as division by zero and invalid tokens
  • Very simple and easy to understand syntax

Sample Code

print 1 + 2 * 3;
print (10 - 5) / 5;
print 2 + 3 * 4;
print (1 + 2) * (3 + 4);
print 10 / 0;    # Error: division by zero

Common Errors

  • Division by zero
  • Invalid token
  • Invalid statement
  • Unexpected token
